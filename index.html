get /member-api/v2/events/{generic_file_identifier}
Premis Events of a generic file (memberApiV2EventsGenericFileIdentifierGet)
List of Premis Events belonging to a particular Generic File. It requires that the user specify the generic_file_identifer as url-encoded, and as part of the base URL.

Path parameters

generic_file_identifier (required)
Path Parameter — Premis Events belonging to a particular Generic File. Requires to specify the generic_file_identifer as url-encoded.

Query parameters

id (optional)
Query Parameter — APTrust's internal ID for an object.
identifier (optional)
Query Parameter — The unique identifier for an event in the APTrust repository. It is generated by APTrust when the event is created.
event_type (optional)
Query Parameter — Refers to the type of Premis Event that was created.
date_time (optional)
Query Parameter — Date when the premis event was created.
detail (optional)
Query Parameter — A brief message explaining the purpose of the premis event generation, tailored to the type of event that was created.
outcome_information (optional)
Query Parameter — Additional information pertinent to the individual process that was run in tandem with the event generation. May be empty.
outcome (optional)
Query Parameter — Outcome of the event
object (optional)
Query Parameter — Name and version of the code base or gem that was responsible for the process run in tandem with the event generation.
agent (optional)
Query Parameter — Source of the object attribute listed above.
institution_id (optional)
Query Parameter — ID of the institution to which the event belongs.
intellectual_object_id (optional)
Query Parameter — ID of the intellectual object to which the event belongs.
generic_file_id (optional)
Query Parameter — ID of the generic file to which the event belongs, if it is associated with a generic file. May be empty.
created_at (optional)
Query Parameter — Time of creation of object
updated_at (optional)
Query Parameter — Time of last update of object
old_uuid (optional)
Query Parameter — Previous identifier of the event if the identifier was a duplicate - something Pharos no longer allows.

Return type

#/definitions/event
