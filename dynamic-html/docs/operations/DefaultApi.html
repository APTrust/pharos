memberApiV2EventsGenericFileIdentifierGet

List of Premis Events belonging to a particular Generic File. It requires that the user specify the generic_file_identifer as url-encoded, and as part of the base URL.

URL

https://demo.aptrust.org/member-api/v2/events/{generic_file_identifier}

HTTP Method

GET

Response Type

#/definitions/event

Parameters

  • Path: generic_file_identifier String

    Premis Events belonging to a particular Generic File. Requires to specify the generic_file_identifer as url-encoded.

  • Query: id Integer

    APTrust's internal ID for an object.

  • Query: identifier String

    The unique identifier for an event in the APTrust repository. It is generated by APTrust when the event is created.

  • Query: event_type String

    Refers to the type of Premis Event that was created.

  • Query: date_time String

    Date when the premis event was created.

  • Query: detail String

    A brief message explaining the purpose of the premis event generation, tailored to the type of event that was created.

  • Query: outcome_information String

    Additional information pertinent to the individual process that was run in tandem with the event generation. May be empty.

  • Query: outcome String

    Outcome of the event

  • Query: object String

    Name and version of the code base or gem that was responsible for the process run in tandem with the event generation.

  • Query: agent String

    Source of the object attribute listed above.

  • Query: institution_id String

    ID of the institution to which the event belongs.

  • Query: intellectual_object_id String

    ID of the intellectual object to which the event belongs.

  • Query: generic_file_id String

    ID of the generic file to which the event belongs, if it is associated with a generic file. May be empty.

  • Query: created_at String

    Time of creation of object

  • Query: updated_at String

    Time of last update of object

  • Query: old_uuid String

    Previous identifier of the event if the identifier was a duplicate - something Pharos no longer allows.
